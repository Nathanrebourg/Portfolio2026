📊 Dashboard Entreprise

À propos du projet

Application web interne développée pour faciliter la visualisation et l'analyse des données d'entreprise. Cet outil de décisionnel offre une interface intuitive et réactive pour suivre les métriques clés en temps réel.

Stack Technologique

React 18.2
Chart.js
Node.js / Express
MongoDB
Tailwind CSS
REST API
JWT Auth
Responsive Design

Fonctionnalités principales

📈 Graphiques & Visualisation

Graphiques interactifs avec Chart.js pour analyser les tendances, le chiffre d'affaires, les performances client et projections futures.

🔐 Authentification Sécurisée

Système de login avec JWT, sessions utilisateur et gestion des droits d'accès par rôle (admin, manager, employé).

⚡ Performance & Temps réel

Mise à jour automatique des données, cache optimisé et requêtes API asynchrones pour une expérience fluide.

📱 Responsive Total

Fonctionne parfaitement sur desktop, tablette et mobile avec interface adaptable.

Captures d'écran & Démo

Résultats & Impact

Ce dashboard a permis à l'équipe management de gagner 3-4 heures par semaine en automatisant le reporting. Les décisions données sont maintenant basées sur des données à jour et des visualisations claires.
