🎨 Ce Site Portfolio

Concept & Vision

Ce portfolio est un showcase personnel qui reflète mon passion pour le design premium et les expériences visuelles raffinées. Inspiré de l'univers automobile de luxe (Porsche, Ferrari), il combine élégance minimaliste et animations subtiles pour créer une impression lasting.

Principes de Design

🎯 Premium & Minimaliste

Palette de couleurs luxe (noir, or, rouge), typographie élégante (Playfair Display), espace blanc généreux.

⚡ Animations Subtiles

Transitions fluides, effets hover doux, scroll behaviors pour une UX premium sans surcharge.

📱 Full Responsive

Mobile-first design avec breakpoints optimisés pour tablettes et desktop. Navigation burger fluide.

♿ Accessible & Performant

Contraste de couleurs respecté, sémantique HTML, vitesse de chargement optimisée, lighthouse score 95+.

Stack Technologique

Frontend : HTML5 / CSS3 / Vanilla JavaScript • Playfair Display / Inter fonts • CSS Grid & Flexbox • Formspree pour les emails
Hébergement : GitHub Pages ou Netlify (gratuit) • Auto-déploiement à chaque push

Fonctionnalités

✅ Navigation fluide (smooth scroll)
✅ Formulaire de contact fonctionnel (Formspree)
✅ Burger menu responsive
✅ Header sticky avec blur effect

✅ Meta tags & SEO de base
✅ Open Graph pour réseaux sociaux
